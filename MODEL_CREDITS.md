## How to update this file

When adding a new `.p3d` to the public repository, add a row to the relevant table in the same PR. Include enough information that someone forking or auditing the project years from now can trace where the model came from.

If the licensing of an existing model changes (e.g. a previously protected model is open-sourced with the original creator's written agreement per [DEV_LICENSE.md](DEV_LICENSE.md)), move its row to the appropriate table and note the date and authorising creator.
