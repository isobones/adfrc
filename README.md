## About

A continuation of the *Australian Defence Force Uncut* mod from years ago, **ADF Re-Cut** is an authentic and polished Arma 3 mod representing the Australian Defence Force.

## What's in this repo

This repository contains the open-source side of ADF: Re-Cut:

- All configs and scripts
- Textures
- Open-source `.p3d` models — either contributed under APL-SA from other community mods, or models the team has chosen to open-source

The repository **does not contain** the team's protected legacy models (see [ASSETS_LICENSE.md](ASSETS_LICENSE.md) for what that covers and why). The full mod with all assets lives on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=2971219389).

## Contributing

This is a community-driven project. See [CONTRIBUTING.md](CONTRIBUTING.md) for how to get involved, including the local testing workflow.

## Building

Builds are currently produced with Mikero's tools (PboProject) by the maintainer. HEMTT migration is planned.

Contributors generally don't build the full mod themselves; See CONTRIBUTING.md for the local testing flow which avoids the need for a full build.
